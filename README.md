# FinPilot 🐷💳

FinPilot é um aplicativo de finanças pessoais baseado em Open Finance que atua como um copiloto financeiro inteligente.

Diferente de aplicativos tradicionais de controle financeiro, o FinPilot não apenas registra movimentações. Ele analisa, prevê e auxilia na tomada de decisões financeiras.

## Objetivos

- Centralizar informações financeiras
- Sincronizar contas via Open Finance
- Automatizar categorização de despesas
- Projetar fluxo de caixa futuro
- Simular decisões financeiras
- Acompanhar metas financeiras ("Porquinhos")
- Gerar insights personalizados
- Monitorar saúde financeira

## Principais funcionalidades

### Open Finance

- Sincronização automática
- Contas
- Cartões
- Faturas
- PIX
- Empréstimos
- Investimentos

### Dashboard Inteligente

- Saldo disponível
- Fluxo previsto
- Próximos vencimentos
- Alertas

### Cartões

- Controle de faturas
- Parcelamentos
- Limites
- Fechamento
- Vencimento

### Caixa de Entrada Financeira

- Revisão de novas movimentações
- Correção de categorias
- Renomeação de estabelecimentos
- Associação a objetivos financeiros

### Porquinhos

- Metas
- Progresso
- Aportes
- Previsão automática
- Sugestão de economia

### Simulador "E se?"

Permite simular compras antes de realizá-las e visualizar seu impacto financeiro futuro.

### Saúde Financeira

Avaliação automática baseada em diversos indicadores financeiros.

## Tecnologias
